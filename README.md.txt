# 🐧 Palmer Penguins Streamlit App

This is a Streamlit web app for data visualization and analysis of the **Palmer Penguins dataset**.

## 🚀 Features
- Load and display the dataset.
- Show basic statistical insights.
- Interactive visualizations using Seaborn.
